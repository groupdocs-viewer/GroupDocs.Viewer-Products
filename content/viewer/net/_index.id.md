        content: |
          Dengan menggunakan GroupDocs.Viewer API, Anda dapat menampilkan lebih dari 170 format dokumen secara efisien dan andal dengan integritas konten dan struktur dokumen yang utuh. Kode contoh berikut menunjukkan betapa mudahnya melihat representasi HTML dari dokumen DOCX:

          ```cs
          // Buat instance penampil
          using (Viewer viewer = new Viewer("sample.docx")
          {
              // Setel opsi tampilan
              HtmlViewOptions options = HtmlViewOptions.ForEmbeddedResources();
              // Konversi file ke HTML dengan sumber daya yang disematkan
              viewer.View(options);
          }
          ```
